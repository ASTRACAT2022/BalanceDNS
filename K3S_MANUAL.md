# Руководство по развертыванию AstracatDNS в K3s

Это руководство поможет вам перенести AstracatDNS с использования `systemctl` (systemd) на Kubernetes (K3s).

## Предварительные требования

*   Установленный [K3s](https://k3s.io/) или доступ к кластеру Kubernetes.
*   Установленный Docker (для сборки образа).
*   Файлы конфигурации (`config.yaml`, `hosts`) и сертификаты (в папке `cert/`), если используется TLS.

## Быстрый старт

Мы подготовили скрипт `deploy_k3s.sh`, который максимально упрощает процесс миграции. Он делает следующее:
1.  Собирает Docker-образ из текущего кода.
2.  Импортирует образ в K3s (чтобы не нужен был внешний реестр).
3.  Создает конфигурацию в Kubernetes на основе ваших локальных файлов `config.yaml` и `hosts`.
4.  Создает секреты для SSL-сертификатов (если они есть в папке `cert/`).
5.  Запускает приложение.

### Инструкция по запуску

1.  Убедитесь, что вы находитесь в корне проекта.
2.  Запустите скрипт развертывания:
    ```bash
    ./deploy_k3s.sh
    ```
3.  Дождитесь окончания установки. Скрипт покажет статус Pod'ов и IP-адрес сервиса.

## Обновление конфигурации

Если вы изменили `config.yaml` или `hosts`:
1.  Отредактируйте файлы локально.
2.  Снова запустите `./deploy_k3s.sh`. Скрипт обновит ConfigMap и перезапустит Pod'ы для применения изменений.

## Структура Kubernetes

Файлы манифестов находятся в папке `k8s/`:
*   `deployment.yaml`: Описывает, как запускать контейнер, монтировать конфиги и ограничивать ресурсы.
*   `service.yaml`: Открывает порты (DNS: 53 UDP/TCP, DoT: 853, DoH: 443, Metrics: 9090, Admin: 8080).

## Полезные команды

*   **Посмотреть логи:**
    ```bash
    kubectl logs -l app=astracat-dns -f
    ```
*   **Проверить статус:**
    ```bash
    kubectl get pods -l app=astracat-dns
    ```
*   **Удалить развертывание:**
    ```bash
    kubectl delete -f k8s/
    ```

## Миграция с systemd

Вам не нужно переписывать конфиги вручную. Скрипт `deploy_k3s.sh` берет ваш текущий рабочий `config.yaml`, который использовался для systemd, и автоматически переносит его в кластер K3s.
