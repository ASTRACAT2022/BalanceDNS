# 🌐 Astracat DNS Resolver

🚀 **Полноценный рекурсивный DNS resolver**, использующий библиотеку [github.com/nsmithuk/resolver](https://github.com/nsmithuk/resolver), созданный для **высокой производительности** с поддержкой **DNSSEC** и интеллектуальным кэшированием.

## ✨ Особенности

- ⚡ **Высокая производительность**: Обработка **800+ QPS** благодаря пулу воркеров и оптимизированному кэшированию
- 🔐 **DNSSEC валидация**: Полная поддержка проверки цифровых подписей DNS
- 🧠 **Интеллектуальное кэширование**:
  - Двухуровневый кэш с учетом TTL записей
  - Отдельное хранение информации о DNSSEC аутентификации
  - Автоматическая очистка устаревших записей
- 🌐 **Поддержка UDP и TCP**: Работа по обоим протоколам транспорта
- 🧵 **Пул воркеров**: Фиксированное количество горутин (1000) для обработки запросов
- 🛡️ **Защита от перегрузки**: Очередь запросов с отказом при достижении лимита
- 📉 **Минимальное логирование**: Оптимизировано для высокой нагрузки

## 📋 Требования

- Go 1.19+
- Доступ к интернету для рекурсивного разрешения DNS запросов
